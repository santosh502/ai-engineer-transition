## 📚 Learning Phases

### Phase 1: Foundations (Months 1–2)
- [ ] Linear algebra (3Blue1Brown + Khan Academy)
- [ ] Probability & statistics (StatQuest)
- [ ] Python for ML (NumPy, Pandas, Matplotlib, Scikit-learn)
- [ ] ML fundamentals overview (Andrew Ng Week 1-2)
- **Project**: Kaggle EDA notebook

### Phase 2: Machine Learning (Months 2–3)
- [ ] Supervised learning (trees, boosting, SVM)
- [ ] Unsupervised learning (clustering, PCA)
- [ ] Feature engineering
- [ ] Model evaluation (CV, metrics, error analysis)
- [ ] FastAPI deployment
- **Project**: Classification model + API

### Phase 3: Deep Learning (Months 3–4)
- [ ] Neural network fundamentals
- [ ] CNNs (convolutions, transfer learning, ResNet)
- [ ] RNNs & LSTMs
- [ ] Transformers architecture (self-attention, positional encoding)
- [ ] PyTorch hands-on
- **Project**: Build Transformer from scratch

### Phase 4: Generative AI (Months 4–5)
- [ ] LLM lifecycle (pretraining, instruction tuning, RLHF)
- [ ] Prompt engineering (system prompts, few-shot, structured output)
- [ ] Embeddings & semantic search
- [ ] Vector databases (Chroma, Pinecone)
- [ ] Fine-tuning (LoRA, QLoRA)
- **Project**: Semantic search engine

### Phase 5: Production AI Systems (Months 5–8)
- [ ] RAG systems (chunking, retrieval, generation, evaluation)
- [ ] Advanced RAG (hybrid search, re-ranking, HyDE)
- [ ] AI Agents (ReAct, tool calling, LangGraph)
- [ ] MLOps (MLflow, monitoring, CI/CD)
- [ ] LLMOps (cost, reliability, evaluation)
- **Projects**: RAG system + Multi-agent system + Capstone

### Phase 6: Interview Prep (Months 8–15)
- [ ] Portfolio polish (tests, docs, GitHub)
- [ ] System design mocks (2–3 per week, Months 10–12)
- [ ] Coding rounds (1 per week, Months 10–12)
- [ ] Deep dives (transformers, evaluation, scaling, reliability)
- [ ] Job applications & interviews
